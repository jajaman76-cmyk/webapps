영상 프롬프트 생성기

이미지 + 대본 업로드 → 영상 제작용 한국어 프롬프트 자동 생성

💡 API 키는 https://ai.google.dev/ 에서 무료로 생성할 수 있습니다. 저장 버튼을 누르면 다음에도 자동으로 불러옵니다.
모든 프롬프트에 고정 포함 → 업로드된 이미지의 스타일 그대로 유지. 자막넣기 금지.
클릭하거나 이미지를 드래그하세요
파일명을 001.png, 002.png 순서로 저장하면 자동 정렬됩니다
📌 빈 줄(엔터 두 번) = 장면 구분선
한 단락(여러 줄 가능) = 이미지 1장과 자동 매칭
001.png → 1번째 단락  |  002.png → 2번째 단락
단락 수가 이미지보다 적으면 남은 이미지는 대본 없이 생성됩니다
